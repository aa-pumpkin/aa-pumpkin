## Projects

### UnivIS Planner

Timetable planner for Computer Science students at CAU Kiel.

Loads public UnivIS course data, maps lectures and exercises to modules, and generates timetable variants with minimal conflicts.  
Runs locally in the browser without an account or CAU credentials.

**Tech:** Vue, TypeScript, Vite, Pinia, Vitest  
**Repository:** [aa-pumpkin/univis-planner](https://github.com/aa-pumpkin/univis-planner)

### IoT-NDN Privacy Prototype

Bachelor thesis prototype for selective privacy protection in IoT-NDN communication on ESP-based devices.

**Tech:** C++, ESP, NDNph, PlatformIO, Python, HMAC-SHA256

### University Canteen Data Analysis

Data analysis project based on German university canteen data.

**Tech:** Python, pandas, GeoPandas, Streamlit, Altair, Plotly, SQLite, DVC, Docker

### RegNet

University team project for an internal registration system.

**Tech:** Vue, Quasar, JavaScript
