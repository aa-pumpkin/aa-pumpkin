## Projects

### UnivIS Planner — in development

A timetable planner for Computer Science students at CAU Kiel.

The project loads public UnivIS course data, maps lectures and exercises to modules, and generates multiple timetable variants with as few conflicts as possible. It runs locally in the browser without an account or CAU credentials.

**Tech:** Vue, TypeScript, Vite, Pinia, Vitest  
**Repository:** [aa-pumpkin/univis-planner](https://github.com/aa-pumpkin/univis-planner)

### IoT-NDN Privacy Prototype

Bachelor's thesis prototype for selective privacy protection in IoT-NDN communication on ESP-based devices.

**Tech:** C++, ESP, NDNph, PlatformIO, Python, HMAC-SHA256

### University Canteen Data Analysis

University data analysis project based on German university canteen data.

**Tech:** Python, pandas, GeoPandas, Streamlit, Altair, Plotly, SQLite, DVC, Docker

### RegNet

University team project for an internal registration system.

**Tech:** Vue, Quasar, JavaScript
